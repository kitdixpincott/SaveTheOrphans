How to use

As with all other jQeury plugins you'll need to download the savetheorphans.js file and link it up like we have to this file:

<script src="js/savetheorphans.js"></script>

You'll then need to add this into your jQuery:

$(document).ready(function(){
    $("p").savetheorphans({
        orphansToSave: 1
    });
});

Examples

Chaining selectors

You can chain the selectors, so it can work with more than just paragraphs. This will work on p, li and blockquote:

$("p,li,blockquote").savetheorphans({
    orphansToSave: 1
});

Save more orphans!

You can increase the number of orphans to save! For example, upping the orphansToSave variable to 3 will make sure that 4 words are shown on the last line of a paragraph. The following code will always show 3 words on the last line of a paragraph:

$("p").savetheorphans({
    orphansToSave: 2
});

That's all folks

So there we have it. A tiny little plugin to prevent orphaned text on your website.

Enjoy!
